# facial-recognition

> A simple facial recognition script using OpenCV

##  Installation
```
pip install
```

### Clone
- Clone this repo to your local machine using `https://github.com/gurveerdhindsa/facial-recognition.git`

### Run
```
python main.py
```

##  Tech stack
- Python (OpenCV)
